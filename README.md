# Agenda de Contactos en Python

## Descripción
Aplicación de consola desarrollada en Python que permite gestionar una
agenda de contactos, integra CRUD (agregar, editar y eliminar
contactos) además de un buscador por nombre o teléfono.

## Estructura del proyecto

- `contacto.py`: Contiene la clase Contacto.
- `agenda.py`: Contiene la clase Agenda y la lógica de gestión.
- `main.py`: Menú principal e interacción con el usuario.
- `pruebas.py`: Pruebas unitarias del sistema.
- `informe_pruebas.md`: Detalle de las pruebas realizadas.

## Arquitectura
El proyecto sigue un enfoque de Programación Orientada a Objetos (POO),
donde:
- Contacto representa la entidad principal.
- Agenda actúa como gestor de los contactos.

## Requisitos
- Python 3.8 o superior
